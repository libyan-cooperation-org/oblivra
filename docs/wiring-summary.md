# Wiring Summary — Desktop and Web

Date: 2026-04-29

## Desktop (Wails)
- Entrypoint: `main_gui.go` — `runGUI()` creates the Wails app and binds services.
- Backend binding root: `internal/app/app.go` — `App` struct, `New()`, and `wireServices()` map many services to Wails bindings.
- Frontend bridge: `frontend/src/lib/bridge.ts` loads `@wailsio/runtime` and provides event/RPC helpers for desktop.
- UI wiring: `frontend/src/App.svelte` registers menu/tray/window handlers and initializes app stores/routes.
- Wired services: Host, SSH, Vault, Session, Settings, SIEM, Alerting, WindowService, PluginService, Telemetry, AIService, and many more are bound and exposed to the frontend (see `App` fields and `wireServices()`).

## Web (Browser)
- Entrypoints: `frontend-web` (pure web Svelte app) and `frontend` (Svelte app used by Wails for desktop).
- Browser bridge: when running in browser mode the bridge falls back to a WebSocket event stream (`/api/v1/events`) and REST endpoints; `initBrowserBridge()` is in `frontend/src/lib/bridge.ts`.
- Wired UI: the Svelte UI mounts in `frontend/src/main.ts` / `frontend-web/src/main.ts` and `App.svelte` contains the route map and feature initialization.

## Gaps / Not wired / Caveats
- `RunOsquery()` in `internal/app/app.go` returns a not-implemented error (osquery integration planned but not available).
- The old `shell` subsystem was removed (noted in `App.svelte` comments — Phase 33); `/shell` route is offline.
- Native OS features (system tray, native app menu, WindowService pop-outs) are desktop-only; in browser mode they are no-ops or require server-side alternatives.
- Wails-generated RPC modules (the `@wailsjs/...` bindings) are only present in the desktop build; browser builds rely on the backend server and WebSocket/REST endpoints.
- Many features are flagged as phase-based or "planned" in comments; bindings exist so the frontend can call them, but underlying service implementations may be partial or stubbed — you must inspect each `services/*` implementation for completeness.
